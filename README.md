$ pkg install python python2
$ pip2 install requests mechanize bs4
$ pip install requests mechanize bs4
$ git clone https://github.com/N16HT-W4RR10R/Bruteforce-Facebook

$ python2 Brutetarget.py
$ python3 Bruteid.py
